# PRODIGY_FS_01 — Secure User Authentication (Task-01)

This is my **Full-Stack Internship Task-01** project:  
A secure user authentication system using **Spring Boot, JWT, and Email Verification**.

## Features
- User registration & login
- Password hashing with BCrypt
- JWT authentication for protected routes
- Email verification system

## Tech Stack
- Java 17, Spring Boot, Spring Security
- JPA / Hibernate
- Gradle
- PostgreSQL (or H2 for testing)
- JWT (JSON Web Token)
- JavaMail (SMTP for email verification)

./gradlew bootRun
